
# 🌍 COVID-19 Global Analysis

## 📌 Overview
This project analyzes global COVID-19 spread using real-world data.

## 📊 Dataset
- Source: Johns Hopkins University
- Time-series data across 180+ countries

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn / Plotly

## 📈 Visualizations
- Choropleth Map
- Bubble Map
- Histogram + KDE
- Box Plot
- Interactive Time Series

## 🚀 Key Insights
- COVID spread varies by region
- High concentration in specific countries
- Trends change over time

## 📂 Files
- Project3.ipynb → Analysis code
- Presentation.pptx → Slides

## ▶️ How to Run
Open the notebook and run all cells.
